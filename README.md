# 🛒 ByteBazaar (E-commerce Web App)

A modern full-stack e-commerce web application built using **Node.js, Express, and EJS**, designed to showcase dynamic content rendering and clean UI structure.
   
---

## 🚀 Features

* 🛍️ Product listing and display 
* 🔄 Dynamic routing using Express
* 🎨 EJS templating for server-side rendering
* 📁 Organized MVC-like folder structure
* 💡 Clean and responsive UI design

## 🛠️ Tech Stack

* **Backend:** Node.js, Express.js
* **Frontend:** EJS, HTML, CSS
* **Other Tools:** npm, Git

## 📂 Project Structure

```
├── Public/        # Static assets (CSS, JS, Images)
├── Views/         # EJS templates
├── node_modules/  # Dependencies (ignored in Git)
├── app.js         # Main server file
├── package.json   # Project metadata & dependencies
```

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Jasbir001/ByteBazaar
```

2. Navigate into the project directory:

cd ByteBazaar


3. Install dependencies:

npm install

4. Run the application:

node app.js
npm run dev


5. Open in browser:

http://localhost:8123
